Project Overview

Slayer Suite is a comprehensive, modular signage workflow platform designed to manage every aspect of signage projects from initial survey to final installation. Built with modern ES6 modules and a unified framework, it provides professional tools for signage companies to streamline their operations.

Key Features & Innovations

Unified User Interface

  • Consistent header navigation across all apps
  • Persistent user menu with profile settings
  • Global save/load functionality

Design Slayer Highlights

  • Lazy-loaded DOM queries using Proxy pattern for performance
  • Real-time 3D visualization with Three.js integration
  • Advanced grid snapping with customizable units (inches/mm)
  • Layer linking system for grouped movements
  • Auto-dimensioning with visual dimension lines
  • Material-based rendering in 3D view

Mapping Slayer Highlights

  • PDF.js integration for high-quality PDF rendering
  • Custom marker system with 20+ marker types
  • Real-time measurements with scale calibration
  • Photo attachment to markers
  • Export capabilities to JPG and annotated PDF

Getting Started

Running the Application

  1. Open index.html with a live server (VS Code Live Server recommended)
  2. The suite will initialize and display the welcome screen
  3. Click any app button in the header to switch between applications
  4. Use the SLAYER SUITE logo to return to the welcome screen

Key Interactions

Design Slayer

Drag layers from sidebar to canvas, use grid snapping, view in 3D

Mapping Slayer

Upload PDF, place markers, add measurements, export annotated map

Save/Load

Use header buttons to save and load project data across apps

App Switching

Click app buttons to switch, state is preserved between switches
